# Обоюдно — Импро-шоу на ваше мероприятие

## Что это
Продающий сайт-одностраничник для команды по импровизации «Обоюдно».
Аудитория: HR и ивент-менеджеры корпоратов.
Деплой: Vercel (drag-and-drop папки).

## Файлы
```
index.html   — весь сайт, один файл, CSS и JS внутри
CLAUDE.md    — этот файл
```

## Стек
- Чистый HTML/CSS/JS, без фреймворков и сборки
- Шрифты: Cormorant Garamond (заголовки) + Outfit (тело) — Google Fonts CDN
- Анимации: IntersectionObserver, классы .reveal / .visible

## Дизайн-система
```
--black:   #0C0C0C
--dark:    #141414
--gold:    #C9A042
--gold-lt: #E2BB6A
--cream:   #F2EDE4
--muted:   #6B5E4E
--text:    #1A1A1A
```
Секции чередуются: тёмный фон (#dark) → светлый (#cream) → тёмный...

## Структура страницы (сверху вниз)
1. **nav** — фиксированный, при скролле появляется фон
2. **#hero** — «Импро-шоу на ваше мероприятие» + 3 блока (корпоратив / тимбилдинг / частное событие) + CTA
3. **#about** — статы команды (30+, 100+, 7) + блок про Андрея Смирнова
4. **#how** — 4 шага как работает шоу
5. **#guarantees** — 6 карточек гарантий
6. **#case** — кейс корпоратива 2024 (МК + шоу, 100к)
7. **#media-section** — 3 фото-заглушки + 2 видео-ссылки на Яндекс.Диск
8. **#formats** — 3 формата с ценами (25к / 60к / 100к) + CTA «Получить консультацию»
9. **#contact** — Telegram + Email + видео по запросу
10. **footer**

## Как деплоить на Vercel
Перетащить папку с `index.html` на vercel.com/new → Import → выбрать папку.
Файл должен называться именно `index.html`.
